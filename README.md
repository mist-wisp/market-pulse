# 市场洞察 · Market Pulse

实时市场行情分析网站，包含板块拥挤度、估值分位、恐慌/贪婪指数、资金流向等可视化模块。

## 数据源

| 数据 | 来源 | 备注 |
|------|------|------|
| 大盘指数 | 腾讯行情 | 通过服务端代理 |
| 行业板块 | 新浪财经 | 49个行业板块 |
| 概念板块 | 新浪财经 | 175个概念板块 |
| 资金流向 | 东方财富 | 板块主力净流入 |

## 部署

本项目部署在 Vercel，使用 Serverless Functions 代理数据 API。

## 本地开发

```bash
npm i -g vercel
vercel dev
```

访问 http://localhost:3000
